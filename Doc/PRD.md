# IOTWave 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
**IOTWave** - IoT 波形显示框架

### 1.2 产品定位
一个基于 .NET 8.0/.NET 10.0 和 Avalonia UI 的高性能 IoT 波形显示控件库，用于工业物联网场景的实时数据可视化。

### 1.3 目标用户
- 工业 IoT 开发者
- 数据可视化工程师
- 自动化系统开发者

---

## 2. 核心功能需求

### 2.1 波形显示组件

| 模块 | 功能描述 | 优先级 |
|------|---------|--------|
| `WaveListPanel` | 主容器控件，管理多个曲线面板 | P0 |
| `CurvePanel2` | 单个曲线面板，支持多曲线叠加显示 | P0 |
| `StatusPanel` | 状态面板，用于显示离散状态数据 | P1 |
| `IOTChart` | 整合波形图表控件 | P0 |

### 2.2 标记功能

| 标记类型 | 功能描述 | 优先级 |
|---------|---------|--------|
| `TimeMarker` | 时间标记线（垂直虚线 + 标签） | P1 |
| `TimeRangeMarker` | 时间范围标记（半透明区域 + 标签） | P1 |
| `YMarker` | Y 轴标记 | P1 |
| `Cursor` | 光标线（实时时间显示） | P0 |

### 2.3 交互功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 缩放 | Y 轴缩放（鼠标滚轮） | P0 |
| 平移 | Y 轴拖拽平移 | P0 |
| 光标追踪 | 鼠标移动显示时间 | P0 |
| 图例交互 | 曲线显示/隐藏切换 | P1 |

### 2.4 样式定制

| 功能 | 描述 | 状态 |
|------|------|------|
| 默认模板 | CurvePanel2 内置默认模板 | ✅ 已实现 |
| 自定义模板 | 使用时可重定义 Template | ✅ 已实现 |
| 图例样式 | 支持 `curveLegend` / `curveLegendWithValue` 两种样式 | ✅ 已实现 |

## 3. 数据模型

### 3.1 核心模型

```
Models/
├── CurveData.cs          # 曲线数据点集合
├── CurveGroup.cs         # 曲线组（包含多条曲线）
├── TimePoint.cs          # 时间点数据
├── TimeMarker.cs         # 时间标记
├── TimeRangeMarker.cs    # 时间范围标记
├── YMarker.cs            # Y 轴标记
├── DataSeriesGroupBase   # 数据序列基类
└── StatuSeriesGroup      # 状态序列组
```

---

## 4. 非功能需求

### 4.1 性能要求
- 支持高频数据更新（≥40点/秒）
- 大数据量渲染优化（虚拟化）
- 低延迟响应（<16ms）

### 4.2 兼容性
- .NET 8.0 / .NET 10.0
- 跨平台支持

## 6. 待开发功能（建议）

添加统计值显示功能，如一天内的最大值，最小值，平均值；每个统计值对应CurveGroup 内的一条曲线，统计值相对TimePoint添加一天内发生的具体时间，如最大值发生在9点00，实现该显示，并在MainWindow添加一个TabPage 展示统计值。要求能显示光标处统计值的发生时刻及统计值

## 7. 技术架构

### 7.1 技术栈
- **框架**: .NET 8.0 / .NET 10.0
- **UI 框架**: Avalonia UI
- **架构模式**: MVVM

### 7.2 项目结构

```
IOTWave/
├── IOTWave/                 # 核心库

│   ├── Models/             # 数据模型

│   └── Themes.axaml        # 主题样式
├── IOTWaveDemo/            # 演示项目
├── IOTWave.Tests/          # 单元测试
└── Doc/                    # 文档
```
