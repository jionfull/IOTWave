# IOTWave 项目总结报告

## 项目概述
IOTWave是一个基于 .NET 8.0 和 .NET 10 的物联网(IoT)波形显示和数据处理框架库。该项目提供高性能的波形显示控件、时间轴标记功能和灵活的数据可视化能力。

## 项目结构
```
IOTWave/
├── IOTWave/                 # 核心库
│   ├── Models/             # 数据模型
│   │   ├── CurveData.cs    # 曲线数据模型
│   │   ├── CurveGroup.cs   # 曲线组模型
│   │   ├── TimePoint.cs    # 时间点模型
│   │   ├── TimeMarker.cs   # 时间标记
│   │   ├── TimeRangeMarker # 时间范围标记
│   │   └── YMarker.cs      # Y轴标记
│   ├── Views/              # Avalonia UI 控件
│   │   ├── CurvePanel.axaml.cs  # 曲线面板
│   │   ├── WaveListPanel.axaml.cs # 波形列表面板
│   │   ├── TimeAxis.cs     # 时间轴控件
│   │   ├── YAxisRenderer.cs # Y轴渲染器
│   │   └── IChartGlobal.cs # 图表全局接口
│   └── ViewModels/         # 视图模型
├── IOTWaveDemo/            # 演示项目
├── IOTWave.Tests/          # 单元测试
└── Doc/                    # 文档
```

## 核心功能

### 波形显示
- 高性能波形渲染引擎
- 支持大量数据点的实时显示
- 智能数据下采样算法，保持视觉质量的同时提升性能

### 交互功能
- 鼠标滚轮时间轴缩放
- 拖拽时间轴平移
- Ctrl+滚轮 Y轴缩放
- Ctrl+拖拽 Y轴平移
- 数据点高亮显示

### 数据管理
- TimePoint 时间点模型
- CurveData 曲线数据封装
- CurveGroup 曲线组管理
- 支持多曲线同屏显示

### 标记系统
- TimeMarker 时间点标记
- TimeRangeMarker 时间范围标记
- YMarker Y轴水平标记

### 性能优化
- 智能数据点下采样
- 边界点保留算法
- 极值保留算法
- UI元素缓存机制
